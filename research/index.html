Research Experience

Touch Sensing for Soft-stuffed Robots

In this part, we developed a distributed, soft and flexible touch sensing system which conforms which conforms with the softness of soft-stuffed robots.
For more information, please see the reference publication [1] in Publications.

Two-layered Hidden Markov Models for Unsupervised Modeling of Social Touch

In this project, we implemented a two layered hidden Markov model for unsupervised learning of time-series sensing signals. The method has been applied for on social touch interaction samples.
For more information, please see the reference publication [3] in Publications.

Unsupervised Learning for Embrace Pose Recognition Method

In this study, we attempted to create a recognition system which can learn patterns of embrace pose, a way of holding by way of unsupervised learning and utilize the selected model as an embrace recognizer. In addition, we also provided a method for assigning classes of human actions to the selected model’s clusters which can then be used as the recognized output. Finally, based on the quantitative analysis of the method’s performance, we proposed a guideline which could potentially help to select the recognition model with possibility of high performance.
For more information, please see the reference publication [1] and [2] in Publications.

Automated Retinal Hemorrhage Detection for Diabetic Retinopathy

In this project, we attempted to detect hemorrhages, which are the bleeding of the blood vessels within the retina, from the retinal images. These hemorrhages are the sign of diabetic retinopathy, an eye disease which often occurs in people with diabete.
Here, we extracted blob candidates using image preprocessing techniques and then applied rule-based classification to classify hemorrhages.
For more information, please see the reference publication [7] in Publications.
