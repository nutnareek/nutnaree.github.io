Two-layered Hidden Markov Models for Unsupervised Modeling of Social Touch

  • Defined an embrace pose as a way of holding a robot.
  • Developed an onboard soft and flexible touch sensing system using conductive fabrics.
  • Proposed an unsupervised learning approach based on two-layered hidden Markov models (2L-HMMs) for modeling repeating patterns in time-series sensor data.
  • Analyzed repeating patterns of embrace interaction data samples using the upper hidden Makov model’s state machine generated by GraphViz.

For more information, please see the reference publication [3] in Publications.

Embrace-based Social Touch Recognition

In this study, we attempted to create a recognition system which can detect embrace pose, a way of holding, and hand-based touch gestures. Based on the observation that both forms of social touch require different information necessary for detection: touch location for embrace pose and dynamic motion information for touch gestures, we decided to create two separated recognition systems for both forms of social touch.

Embrace Pose Recognition

  • Performed data collection and signal analysis of embrace pose and touch gesture interaction (sensor) data.
  • Proposed an unsupervised approach for automatic categorization of embrace patterns based on k-means algorithm.
  • Provided an approach to assign action label to each cluster without the need for ground truths.
  • Suggested a guideline for selecting a classifier from the k-means models which could potentially provide good performance, based on the quantitative analysis of the performance of the method.
  • Classified embrace poses using the selected class-assigned k-means model.
  • Proposed a supervised method for touch gesture recognition based on multiple filtering of dynamic information and random forest classification.
